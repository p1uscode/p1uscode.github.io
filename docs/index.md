# p1uscode

自作 lab とナレッジの置き場。各 lab は独立して動くように作ってあり、下のリンクから入る。

## Labs

### [ai/](https://p1uscode.com/ai/)

ローカルで動く AI スタック (LLM / エージェント / 観測 / RAG / マルチモーダル) を題材に、**仕組みを手を動かして理解する**ための実験環境。

- **setup** — Docker Compose によるサービス群の立ち上げ
- **hands-on** — Open WebUI / Langfuse / mitmproxy / agent-demo を触りながら観察する演習
- **theory** — LLM エージェントの内部を下の層から順に理解するための座学 (00-18 章)
